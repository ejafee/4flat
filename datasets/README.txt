RESA Internal Datasets Folder
==============================
Place any CSV or Excel (.xlsx) files in this folder.
RESA will automatically read and use them on every analysis request.

Suggested datasets to add:
- Shopee trending products CSV
- Lazada product pricing data
- TikTok Shop sales data
- Malaysian SME sales benchmarks
- Fashion / F&B / beauty market trend data
- Any platform export you want RESA to learn from

No code changes needed. Just drop the file in and RESA uses it immediately.
